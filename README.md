# 🌐 Ayush Katiyar – Full Stack Developer Portfolio

A modern, responsive, and animated **full-stack portfolio website** built to showcase my skills, projects, and experience as a developer.  
Designed with clean UI, smooth animations, and scalability in mind.

🔗 **Live Demo**: https://portfolio-ntvy-three.vercel.app/

---

## ✨ Features

### 🎨 UI & UX
- Modern, clean, and professional design
- Fully responsive (Mobile, Tablet, Desktop)
- Smooth page transitions & animations
- Dark / Light mode toggle
- Interactive hover effects
- Consistent typography and spacing

### 🧭 Navigation
- Sticky responsive navbar
- Active route highlighting
- Smooth scrolling between sections
- Mobile-friendly hamburger menu

### 👤 Sections Included
- **Home** – Intro with animated text & CTA
- **About Me** – Education, background & career goals
- **Skills** – Categorized technical skills with icons
- **Projects** – Detailed project cards with live & GitHub links
- **Experience / Education** – Timeline-based layout
- **Certificates** – Verified certifications showcase
- **Contact** – Functional contact form

### 📬 Contact & Backend
- Working contact form
- Email notifications (can be integrated)
- Form validation
- Scalable backend-ready structure

### ⚡ Performance & SEO
- Fast loading with Vite
- SEO-friendly meta tags
- Optimized assets
- Accessible components

---
